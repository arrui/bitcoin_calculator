# BTC Mining Calculator

比特币挖矿成本计算器

## 功能

- 实时获取 BTC 价格
- 计算挖矿盈亏平衡点
- 对比主流矿机收益
- 支持交互模式和命令行参数

## 使用方法

### 命令行模式

```bash
# 使用默认参数
python3 btc_mining.py

# 指定 BTC 价格
python3 btc_mining.py --price 85000

# 指定电价和矿机
python3 btc_mining.py --electricity 0.35 --machine S19XP

# 指定算力
python3 btc_mining.py --hashrate 5000
```

### 交互模式

```bash
python3 btc_mining.py --interactive
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| --price | BTC 价格 (USD) | 自动获取 |
| --electricity | 电价 (CNY/kWh) | 0.55 |
| --machine | 矿机型号 | S19XP |
| --hashrate | 算力 (TH/s) | 10000 |
| --interactive | 交互模式 | False |

## 矿机参数

| 型号 | 算力 | 功率 | 价格 |
|------|------|------|------|
| S19KPro | 110T | 3250W | ¥8,500 |
| S19XP | 140T | 3250W | ¥9,500 |
| S21XP | 134T | 5300W | ¥12,000 |
| S21Pro | 234T | 7150W | ¥18,000 |
